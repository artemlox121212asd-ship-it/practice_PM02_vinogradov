В ходе пятого дня практики мною разработано графическое приложение на Python с использованием библиотек Tkinter и mysql-connector-python для управления таблицей «Гости» из базы данных «Гостиница». Приложение реализует полный набор CRUD-операций: просмотр всех записей в таблице Treeview, добавление новых гостей, обновление данных выбранного гостя и удаление записей. Для таблицы «Гости» настроены следующие поля: id_гостя (первичный ключ с автоинкрементом, скрыт от редактирования), фамилия (обязательное), имя (обязательное), отчество (необязательное), телефон (обязательное), паспорт (обязательное), email (необязательное). Приложение включает главное окно с рамкой полей ввода, кнопками управления и таблицей с данными; при выборе строки в Treeview поля автоматически заполняются для редактирования. Реализована проверка обязательных полей перед добавлением и обновлением, подтверждение удаления через messagebox, обработка ошибок подключения и выполнения SQL-запросов. Дополнительно добавлена функция поиска по всем текстовым полям (фамилия, имя, телефон, паспорт) и экспорт данных в CSV-файл. Код структурирован в виде класса DatabaseApp с методами connect_db, refresh_table, add_record, update_record, delete_record, search и export_to_csv. В ходе разработки возникли сложности с корректной настройкой Treeview и привязкой события выбора строки, а также с формированием динамического SQL-запроса для обновления, где потребовалось исключить первичный ключ из списка изменяемых полей. Приложение успешно протестировано: добавлено 3 новых гостя, обновлены данные у 2 записей, удалена 1 запись с подтверждением, выполнено несколько поисковых запросов и экспорт данных в CSV. Все файлы (app.py, requirements.txt, README.md, скриншоты главного окна, добавления, обновления и удаления) сохранены в папку Day5 репозитория и закоммичены в Git. Дневник практики заполнен.
