# Отладка программы - Вариант №5

## 📋 Описание
Отладка программы с использованием специализированных средств отладки Python.
Вариант №5: Генераторы, битовые операции, рекурсия и замыкания.

## 🐛 Найденные ошибки

| № | Тип ошибки | Описание |
|---|------------|----------|
| 1 | **StopIteration** | Неправильное количество итераций цикла |
| 2 | **Логическая ошибка** | Неправильный битовый сдвиг вместо умножения |
| 3 | **RecursionError** | Бесконечная рекурсия |
| 4 | **Утечка памяти** | Накопление данных в замыкании и глобальном кеше |
