Цель работы реализовать веб-приложение на Flask с авторизацией разграничением ролей админ и работник защитой от ботов капча и подключением к базе данных Hotel_Variant5

Выполненные задачи установка Flask и зависимостей создание таблицы Пользователи в БД реализация формы входа с капчей хеширование паролей через bcrypt сессии и выход из системы декораторы loginrequired и adminrequired панель администратора с управлением пользователями панель работника с просмотром бронирований управление статусами бронирований

Структура проекта включает apppy createuserspy requirementstxt папку templates с basehtml loginhtml и подпапками admin и worker а также static с stylecss

Тестовые пользователи admin с паролем admin123 роль администратор и worker с паролем worker123 роль работник

Администратор имеет полный доступ просмотр статистики управление пользователями добавление и удаление управление бронированиями изменение статусов Работник может только просматривать бронирования и изменять их статусы

Защита маршрутов реализована через декораторы adminrequired ограничивает доступ к админским разделам loginrequired требует авторизации для любых защищённых страниц

Капча реализована как простое сложение двух случайных чисел пароли хешируются библиотекой bcrypt сессии хранятся на сервере с секретным ключом

Запуск проекта на новом компьютере требует установки Python выполнения pip install r requirementstxt затем python createuserspy для создания пользователей и python apppy для запуска сервера после чего приложение доступно по адресу http1270015000

Стек технологий Flask 230 как вебфреймворк MySQL как база данных flaskmysqldb для подключения bcrypt 401 для хеширования паролей HTML и CSS для фронтенда
